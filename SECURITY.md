# Security Policy

## Reporting a Vulnerability

If you discover a security issue in OriginAI, please report it by opening an issue in this repository and marking it as a security issue. Do not disclose the issue publicly until it has been addressed.

## Supported Versions

This repository supports the current release of OriginAI and is maintained on a best-effort basis.

## Handling

We aim to acknowledge reports within 48 hours and provide a fix or mitigation as soon as possible.
